This project explores student retention patterns among probationary students, using synthetic data modeled after real-world university retention trends.
The goal is to understand how students with different GPA ranges during Fall 2024 progressed in subsequent semesters;  highlighting re-enrollment, dismissal, and reinstatement outcomes.
The analysis combined Python and Excel to perform data cleaning, visualization, and trend interpretation.
